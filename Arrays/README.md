### java에서의 배열
1. 임의의 위치에 값을  넣을 때 O(N) 이다. 

    ==> 넣을 위치를 찾아야 되고, 넣고 난 후 원래 1에 있던 배열부터 1칸씩 뒤로 가야 되기 때문
2. 마찬가지로 임의의 위치의 값을 제거할 때 O(N) 이다.
3. 임의의 위치의 원소를 확인 또는 변경은 O(N) 이다.
4. 배열의 맨 끝에 원소를 추가, 제거하는 것은 O(1) 이다.
